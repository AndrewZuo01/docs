## 获取黑名单列表

</center>

### 简要描述
- 获取指定用户的黑名单列表。
### 请求方式
- `post` 
### 请求URL
- `http://x.x.x.x:10002/friend/get_black_list`

### 请求参数示例


```json
{
  "userID": "11111111",
  "pagination": {
    "pageNumber": 1,
    "showNumber": 100
  }
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|userID|必填|string|用户ID|
|pagination|必填|object|分页参数结构体|
|pagination.pageNumber|必填|int|当前页码，从1开始|
|pagination.showNumber|必填|int|当前页请求数量|

```json
          "ex": ""
        },
        "addSource": 0,
        "operatorUserID": "openIMAdmin",
        "ex": ""
      }
    ],
    "total": 1
  }

```
### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
|blacks|array|[黑名单信息](docs/restapi/commonFields.mdx#blackinfo)列表|
|blacks.ownerUserID|string|用户ID|
|blacks.createTime|int|拉黑时间（毫秒时间戳）|
|blacks.blackUserInfo|object|拉黑的用户信息|
|blacks.blackUserInfo.userID|string|拉黑的用户ID|
|blacks.blackUserInfo.nickname|string|拉黑用户的昵称|
|blacks.blackUserInfo.faceURL|string|拉黑用户的头像|
|blacks.blackUserInfo.ex|string|拉黑用户的扩展字段|
|blacks.addSource|int|拉黑来源|
|blacks.operatorUserID|string|操作者|
|blacks.ex|string|扩展字段|
|total|int|黑名单总数|
### 失败返回示例


```json
{
  "errCode": 1004,
  "errMsg": "RecordNotFoundError",
  "errDlt": "[1004]RecordNotFoundError"
}
```
### 失败返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,具体查看全局错误码文档|
|errMsg|string|错误简要信息|
|errDlt|errDlt|错误详细信息|
