## 获取指定好友双向申请列表

</center>

### 简要描述
- 获取申请列表，包含收到和发送的好友申请
### 请求方式
- `post` 
### 请求URL
- `http://x.x.x.x:10002/friend/get_designated_friend_apply` 


### Header

|operationID|1646445464564|必填|string|operationID用于全局链路追踪|

### 请求参数示例


```json
{
  "fromUserID": "3489932643",
  "toUserID": "5345125128"
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|fromUserID|必填|string|既作为好友申请者，也作为被申请者|
|toUserID|必填|string|既作为好友申请者，也作为被申请者|
