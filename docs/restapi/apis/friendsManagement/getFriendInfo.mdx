## 获取指定好友信息

</center>

### 简要描述
- 获取ownerUserID的指定好友userIDList信息。
### 请求方式
- `post` 
### 请求URL
- `http://x.x.x.x:10002/friend/get_specified_friends_info` 


### Header

|:----    |:-------    |:--- |---|------      |
|operationID|1646445464564|必填|string|operationID用于全局链路追踪|

### 请求参数示例


```json
{
  "ownerUserID": "1269325046",
  "userIDList": [
    "2778222451",
    "7624496010"
  ]
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|ownerUserID|必填|string|获取此用户的好友信息|
|userIDList|必填|array|具体好友的userID列表|
