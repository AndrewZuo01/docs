## 获取好友列表

</center>

### 简要描述
- 获取指定用户的好友列表。
### 请求方式
- `post` 
### 请求URL
- `http://x.x.x.x:10002/friend/get_friend_list`

### 请求参数示例


```json
{
  "userID": "11111111",
  "pagination": {
    "pageNumber": 1,
    "showNumber": 100
  }
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|userID|必填|string|用户ID|
|pagination|必填|object|分页参数结构体|
|pagination.pageNumber|必填|int|当前页码，从1开始|
|pagination.showNumber|必填|int|当前页请求数量|

### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
|friendsInfo|array|[好友信息](docs/restapi/commonFields#friendinfo)列表|
|friendsInfo.ownerUserID|string|用户ID|
|friendsInfo.remark|string|备注|
|friendsInfo.createTime|int|创建时间|
|friendsInfo.friendUser|object|好友用户信息|
|friendsInfo.friendUser.userID|string|好友的用户ID|
|friendsInfo.friendUser.nickname|string|好友的用户名|
|friendsInfo.friendUser.faceURL|string|好友的头像|
|friendsInfo.friendUser.ex|string|好友扩展字段|
|friendsInfo.friendUser.createTime|int|好友的创建时间|
|friendsInfo.friendUser.appMangerLevel|string|好友的管理员等级|
|friendsInfo.friendUser.globalRecvMsgOpt|string|好友设置全局是否接收消息，0为接收，1为不接收|
|friendsInfo.addSource|int|添加来源|
|friendsInfo.operatorUserID|string|操作者ID|
|friendsInfo.ex|string|扩展字段|
|total|int|返回的好友总数|
### 失败返回示例


```json
{
  "errCode": 1004,
  "errMsg": "RecordNotFoundError",
  "errDlt": "[1004]RecordNotFoundError"
}
```
### 失败返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,具体查看全局错误码文档|
|errMsg|string|错误简要信息|
|errDlt|errDlt|错误详细信息|
