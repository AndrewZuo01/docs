## 创建群组

</center>

### 简要描述
- 创建群聊，需指定群主和群成员
### 请求方式
- `post` 
### 请求URL
- `http://x.x.x.x:10002/group/create_group`

### 请求参数示例


```json
{
  "memberUserIDs": [
    "1225441072"
  ],
  "adminUserIDs": [
    "2065939257"
  ],
  "ownerUserID": "1054527962",
  "groupInfo": {
    "groupID": "xadxwr24",
    "groupName": "yourg group name",
    "notification": "notification",
    "introduction": "introduction",
    "faceURL": "faceURL url",
    "ex": "ex",
    "groupType": 2,
    "needVerification": 0,
    "lookMemberInfo": 0,
    "applyMemberFriend": 0
  }
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|memberUserIDs|必填|array|初始化群成员userID列表|
|adminUserIDs|必填|array|管理员userID列表|
|ownerUserID|必填|string|群主ID|
|groupInfo|必填|object|群信息对象|
|groupInfo.groupID|选填|string|群ID，可以外部传入，最大字符为64，如果未指定，由系统自动生成|
|groupInfo.groupName|必填|string|群名称|
|groupInfo.notification|必填|string|群公告|
|groupInfo.introduction|必填|string|群介绍|
|groupInfo.faceURL|必填|string|群头像|
|groupInfo.ex|必填|string|群拓展字段|
|groupInfo.groupType|必填|int|群类型，固定为2，否则会出现参数错误|
|groupInfo.needVerification|必填|int|进群是否需要验证；0、申请加入群需要同意，成员邀请可直接进群1、所有人进群需要验证，除了群主管理员邀请进群2、直接进群|
|groupInfo.lookMemberInfo|必填|int|是否能查看其他群成员信息；0、允许查看群成员信息1、不允许查看群成员信息|
|groupInfo.applyMemberFriend|必填|int|群成员是否能添加好友;0、允许从群成员处添加好友1、不允许添加|

### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
|groupInfo.groupID|string|群ID|
|groupInfo.groupName|string|群名称|
|groupInfo.notification|string|群公告|
|groupInfo.introduction|string|群介绍|
|groupInfo.faceURL|string|群头像|
|groupInfo.ownerUserID|string|群主ID|
|groupInfo.createTime|int|创建时间|
|groupInfo.memberCount|int|群成员数量|
|groupInfo.ex|string|群拓展字段|
|groupInfo.status|int|群状态；0、正常状态1、被封禁（暂未用）2、被解散 3、处于全体禁言状态|
|groupInfo.creatorUserID|string|群创建者ID|
|groupInfo.groupType|int|群类型，固定为2|
|groupInfo.needVerification|int|进群是否需要验证；0、申请加入群需要同意，成员邀请可直接进群1、所有人进群需要验证，除了群主管理员邀请进群2、直接进群|
|groupInfo.lookMemberInfo|int|是否能查看其他群成员信息；0、允许查看群成员信息1、不允许查看群成员信息|
|groupInfo.applyMemberFriend|int|群成员是否能添加好友;0、允许从群成员处添加好友1、不允许添加|
|groupInfo.notificationUpdateTime|int|群公告最后更新时间|
|groupInfo.notificationUserID|string|群公告最后设置用户ID|
### 失败返回示例


```json
{
  "errCode": 1202,
  "errMsg": "GroupIDExisted",
  "errDlt": "group id existed xadxwr24: [1202]GroupIDExisted"
}
```
### 失败返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,具体查看全局错误码文档|
|errMsg|string|错误简要信息|
|errDlt|errDlt|错误详细信息|
