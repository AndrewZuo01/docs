## 设置群组信息

</center>

### 简要描述
- 设置群组信息，字段不为空，代表设置该值。
### 请求方式
- `post` 
### 请求URL

### 请求参数示例


```json
{
  "groupInfoForSet": {
    "groupID": "xadxwr24",
    "groupName": "new name",
    "notification": "new notification",
    "introduction": "new introduction",
    "faceURL": "www.newfaceURL.com",
    "ex": "new ex",
    "needVerification": 1,
    "lookMemberInfo": 1,
    "applyMemberFriend": 1
  }
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|groupInfoForSet|必填|object|群信息对象|
|groupInfoForSet.groupID|必填|string|要设置的群ID|
|groupInfoForSet.groupName|选填|string|要设置的群名|
|groupInfoForSet.notification|选填|string|新群公告|
|groupInfoForSet.introduction|选填|string|新群介绍|
|groupInfoForSet.faceURL|选填|string|新群头像|
|groupInfoForSet.ex|选填|string|新群拓展字段|
|groupInfoForSet.needVerification|选填|int|进群是否需要验证；0、申请加入群需要同意，成员邀请可直接进群1、所有人进群需要验证，除了群主管理员邀请进群2、直接进群|
|groupInfoForSet.lookMemberInfo|选填|int|是否能查看其他群成员信息；0、允许查看群成员信息1、不允许查看群成员信息|
|groupInfoForSet.applyMemberFriend|选填|int|群成员是否能添加好友;0、允许从群成员处添加好友1、不允许添加|
### 成功返回示例


```json
{
  "errCode": 0,
  "errMsg": "",
  "errDlt": ""
}
```

|errDlt|errDlt|错误详细信息,无错误时为空|
