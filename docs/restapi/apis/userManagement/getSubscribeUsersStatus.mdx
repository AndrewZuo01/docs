## 获取所有订阅用户的在线状态

</center>

### 简要描述
- 用户某用户获取所有订阅的用户的在线状态
### 请求方式
- `post` 
### 请求URL
- `{{host}}/user/get_subscribe_users_status` 


### Header
|header名|示例值|选填|类型|说明|
|:----    |:-------    |:--- |---|------      |
|operationID|asdad3qds1|必填|string|operationID用于全局链路追踪|
|token|{{token}}|必填|string|管理员或者用户token|


### 请求参数示例


```json
{
  "userID": "9609746682"
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|userID|必填|string|用户请求账号|
### 成功返回示例


```json
{
    "errCode": 0,
    "errMsg": "",
    "errDlt": "",
    "data": {
        "statusList": [
            {
                "userID": "9168684795",
                "status": 0,
                "platformIDs": null
            }
        ]
    }
}
```
### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
|statusList|array|状态返回结果|
|statusList.userID|string|用户账号|
|statusList.status|int|在线状态，1 - online ; 0 - offline|
|statusList.platformIDs|array|平台号，可能多端在线|
