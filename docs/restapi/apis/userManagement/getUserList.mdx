## 获取用户列表

</center>

### 简要描述
- 获取已注册用户列表。
### 请求方式
- `post` 
### 请求URL
- `http://x.x.x.x:10002/user/get_users`

### 请求参数示例


```json
{
  "pagination": {
    "pageNumber": 1,
    "showNumber": 100
  }
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|pagination|必填|object|分页参数结构体|
|pagination.pageNumber|必填|int|当前页码，从1开始|
|pagination.showNumber|必填|int|当前页请求数量|

### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
|total|int|用户总数|
|users|array|[用户信息](docs/restapi/commonFields.mdx#userinfo)列表|
### 失败返回示例

  "errMsg": "ServerInternalError",

|errMsg|string|错误简要信息|
|errDlt|errDlt|错误详细信息|
