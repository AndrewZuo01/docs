## 普通用户获取用户在线状态

</center>

### 简要描述
- 获取一批用户各平台在线情况。
#### 接口状态
- 开发中
### 请求方式
- `post` 
### 请求URL
- `http://x.x.x.x:10002/user/get_users_status`

### 请求参数示例


```json
{
    "userID": "9168684795",
    "userIDs": [
        "3436560733",

|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|userID|必填|string|请求者|
|userIDs|必填|string|获取的用户状态列表|

    "errMsg": "",

    "data": {

                "platformIDs": null

            {
                "userID": "9168684795",
                "status": 1,
                "platformIDs": [
                    2
                ]
            },
            {
                "userID": "8450535746",
                "status": 0,
                "platformIDs": null
            }
        ]
    }
}
```
### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
|statusList|array|状态返回结果|
|statusList.userID|string|用户账号|
|statusList.status|int|在线状态，1 - online ; 0 - offline|
|statusList.platformIDs|array|平台号，可能多端在线|
