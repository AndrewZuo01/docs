### 请求参数示例


```json
{
  "keyword": "",
  "startTime": 1681996104614,
  "endTime": 1692996104614,
  "pagination": {
    "pagenumber": 1,
    "showNumber": 10
  }
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|

|pagination.showNumber|必填|string|每页数量|

### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
|LogsInfos|array||
|LogsInfos.userID|string|用户ID|
|LogsInfos.platform|string|平台|
|LogsInfos.url|string|日志url|
|LogsInfos.createTime|string|创建时间|
|LogsInfos.nickname|string|用户昵称|
|LogsInfos.logID|string|日志ID|
|LogsInfos.filename|string|日志文件名|
|LogsInfos.systemType|string|系统型号 如安卓3.5|
|LogsInfos.ex|string|扩展字段|
|LogsInfos.version|string|版本号|
|total|string|日志总数|
