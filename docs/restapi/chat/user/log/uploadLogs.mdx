### 请求参数示例


```json
{
  "platform": 1,
  "fileURLs": [
    {
      "filename": "test.txt",
      "url": "www.xxx.com"
    },
    {
      "filename": "test.txt",
      "url": "www.xxx.com"
    }
  ],
  "version": "release 3.1",
  "systemType": "android 3.5",
  "ex": ""
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|platform|必填|string|平台|
|fileURLs|必填|array||
|fileURLs.filename|必填|string|日志文件名|
|fileURLs.url|必填|string|日志url|
|version|必填|string|版本号|
|systemType|选填|string|系统型号 如安卓3.5|
|ex|选填|string|扩展字段|
### 成功返回示例


```json
{
  "errCode": 0,
  "errMsg": "",
  "errDlt": ""
}
```
### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
