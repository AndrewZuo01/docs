### 成功返回示例的参数说明

| 参数名                         | 类型    | 说明                                                             |
| :----------------------------- | :------ | :--------------------------------------------------------------- |
| conversations                  | array   | 会话列表                                                         |
| conversations.ownerUserID      | string  | 会话的拥有者用户 ID                                              |
| conversations.conversationID   | string  | 会话 ID                                                          |
| conversations.recvMsgOpt       | number  | 接收消息参数，0 为接收消息，1 为不接收消息，2 为接收消息但不提醒 |
| conversations.conversationType | string  | 会话类型，1 为单聊，3 为群聊                                     |
| conversations.userID           | string  | 会话用户 ID，会话类型为 1 才有该字段                             |
| conversations.groupID          | string  | 会话的群，会话类型为 3 才有该字段                                |
| conversations.isPinned         | boolean | 是否置顶                                                         |
| conversations.attachedInfo     | string  | openIM 使用的拓展字段                                            |
| conversations.isPrivateChat    | boolean | 是否开启阅后即焚                                                 |
| conversations.groupAtType      | number  | 群会话公告强提示类型，群公告有人@ownerUserID，@全体 特殊标识符   |
| conversations.ex               | string  | 用户使用的拓展字段                                               |
| conversations.burnDuration     | number  | 阅后即焚消息持续时间                                             |
| conversations.minSeq           | number  | 该用户在该会话能拉取到的最小 seq                                 |
| conversations.maxSeq           | number  | 该用户在该会话能拉取到的最大 seq                                 |
