### 请求参数示例

```json
{
  "userIDs": ["kernaltestuid8"],
  "conversation": {
    "userID": "kernaltestuid9",
    "conversationID": "si_kernaltestuid8_kernaltestuid9",
    "conversationType": 1,
    "hasReadSeq": 5,
    "updateUnreadCountTime": 10000000
  }
}
```

| 字段名                             | 选填 | 类型   | 说明                         |
| :--------------------------------- | :--- | :----- | ---------------------------- |
| userIDs                            | 必填 | array  | 要更新会话的用户 ID 列表     |
| conversation                       | 必填 | object | 会话对象                     |
| conversation.userID                | 必填 | string | 会话用户 ID，为群聊时为空    |
| conversation.conversationID        | 必填 | string | 会话群 ID，为单聊时为空      |
| conversation.conversationType      | 必填 | string | 会话类型，1 为单聊，3 为群聊 |
| conversation.hasReadSeq            | 选填 | string |                              |
| conversation.updateUnreadCountTime | 选填 | string |                              |
