### 成功返回示例的参数说明

| 参数名                        | 类型   | 说明               |
| :---------------------------- | :----- | :----------------- |
| blacks                        | array  | 黑名单列表         |

| blacks.createTime             | string | 拉黑时间           |
| blacks.blackUserInfo          | object | 拉黑的用户信息     |
| blacks.blackUserInfo.userID   | string | 拉黑的用户 ID      |
| blacks.blackUserInfo.nickname | string | 拉黑的用户昵称     |
| blacks.blackUserInfo.faceURL  | string | 拉黑的用户头像     |
| blacks.blackUserInfo.gender   | int    | 拉黑的用户性别     |
| blacks.blackUserInfo.ex       | string | 拉黑的用户 ex 字段 |
| blacks.addSource              | int    | 拉黑来源           |
| blacks.operatorUserID         | string | 操作者 ID          |
| blacks.ex                     | string | ex 字段            |
| total                         | string | 总数               |

### 失败返回示例

```json
{
  "errCode": 1004,
  "errMsg": "RecordNotFoundError",
  "errDlt": "[1004]RecordNotFoundError"
}
```

### 失败返回示例的参数说明

| 参数名  | 类型   | 说明                          |
| :------ | :----- | :---------------------------- |
| errCode | int    | 错误码,具体查看全局错误码文档 |
| errMsg  | string | 错误简要信息                  |
| errDlt  | errDlt | 错误详细信息                  |
