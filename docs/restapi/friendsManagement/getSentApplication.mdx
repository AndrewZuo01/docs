### 成功返回示例的参数说明

| 参数名                       | 类型   | 说明             |
| :--------------------------- | :----- | :--------------- |
| friendRequests               | array  | 好友请求信息列表 |
| friendRequests.fromUserID    | string | 申请人 ID        |
| friendRequests.fromNickname  | string | 申请人昵称       |
| friendRequests.fromFaceURL   | string | 申请人头像       |
| friendRequests.fromGender    | int    | 申请人性别       |
| friendRequests.toUserID      | string | 被申请人 ID      |
| friendRequests.toNickname    | string | 被申请人昵称     |
| friendRequests.toFaceURL     | string | 被申请人头像     |
| friendRequests.toGender      | int    | 被申请人性别     |
| friendRequests.handleResult  | int    | 处理结构         |
| friendRequests.reqMsg        | string | 申请信息         |
| friendRequests.createTime    | string | 申请时间         |
| friendRequests.handlerUserID | string | 处理用户 ID      |
| friendRequests.handleMsg     | string | 处理信息         |
| friendRequests.handleTime    | string | 处理时间         |
| friendRequests.ex            | string | ex 字段          |
| total                        | string | 总数             |
