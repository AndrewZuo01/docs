### 成功返回示例的参数说明

| 参数名                       | 类型   | 说明                                  |
| :--------------------------- | :----- | :------------------------------------ |
| FriendRequests               | array  | 收到的的好友申请列表                  |
| FriendRequests.fromUserID    | string | 申请好友的用户 ID                     |
| FriendRequests.fromNickname  | string | 申请好友的用户名                      |
| FriendRequests.fromFaceURL   | string | 申请好友的头像                        |
| FriendRequests.toUserID      | string | 被申请的用户 ID                       |
| FriendRequests.toNickname    | string | 被申请的用户名                        |
| FriendRequests.toFaceURL     | string | 被申请的用户头像                      |
| FriendRequests.handleResult  | string | 申请的结果，0 代表未处理，-1 代表拒绝 |
| FriendRequests.reqMsg        | string | 申请的信息                            |
| FriendRequests.createTime    | string | 申请时间                              |
| FriendRequests.handlerUserID | string | 处理用户的 ID                         |
| FriendRequests.handleMsg     | string | 处理的信息                            |
| FriendRequests.handleTime    | string | 处理的时间                            |
| FriendRequests.ex            | string | 拓展字段                              |
| total                        | string | 总数                                  |

### 失败返回示例

```json
{
  "errCode": 1004,
  "errMsg": "RecordNotFoundError",
  "errDlt": "[1004]RecordNotFoundError"
}
```

### 失败返回示例的参数说明

| 参数名  | 类型   | 说明                          |
| :------ | :----- | :---------------------------- |
| errCode | int    | 错误码,具体查看全局错误码文档 |
| errMsg  | string | 错误简要信息                  |
| errDlt  | errDlt | 错误详细信息                  |
