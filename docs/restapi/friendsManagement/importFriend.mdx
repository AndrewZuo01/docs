### 请求参数示例

```json
{
  "ownerUserID": "608035037",
  "friendUserIDs": ["3702250606", "3853887332"]
}
```

| 字段名        | 选填 | 类型   | 说明                   |
| :------------ | :--- | :----- | ---------------------- |
| ownerUserID   | 必填 | string | 需要导入好友的用户     |
| friendUserIDs | 必填 | string | 导入的好友用户 ID 列表 |
