</center>

### 简要描述

- 导入好友关系，该接口需要使用管理员 token

### 请求方式

- `post`

### 请求 URL

- `{{host}}/friend/import_friend`

### 请求参数示例

```json
{
  "ownerUserID": "1054527962",
  "friendUserIDs": ["2050169792", "3034068043"]
}
```

| 字段名        | 选填 | 类型   | 说明                   |
| :------------ | :--- | :----- | ---------------------- |
| ownerUserID   | 必填 | string | 需要导入好友的用户     |
| friendUserIDs | 必填 | string | 导入的好友用户 ID 列表 |
