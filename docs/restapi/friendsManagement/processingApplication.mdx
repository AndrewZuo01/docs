### 请求参数示例

```json
{
  "fromUserID": "kernaltestuid8",
  "toUserID": "kernaltestuid9",
  "handleResult": 1,
  "handleMsg": "agree"
}
```

| 字段名       | 选填 | 类型   | 说明                        |
| :----------- | :--- | :----- | --------------------------- |
| fromUserID   | 必填 | string | 主动发起的申请者用户 ID     |
| toUserID     | 必填 | string | 被添加好友的用户 ID         |
| handleResult | 必填 | string | 1 为同意申请，-1 为拒绝申请 |
| handleMsg    | 必填 | string | 处理的信息                  |

### 成功返回示例
