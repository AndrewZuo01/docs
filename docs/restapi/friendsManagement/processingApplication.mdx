### 请求参数示例

```json
{
  "fromUserID": "282414670s",
  "toUserID": "3086243450",
  "handleResult": 1,
  "handleMsg": "agree"
}
```

| 字段名       | 必选 | 类型   | 说明                        |
| :----------- | :--- | :----- | --------------------------- |
| fromUserID   | 1    | string | 主动发起的申请者用户 ID     |
| toUserID     | 1    | string | 被添加好友的用户 ID         |
| handleResult | 1    | string | 1 为同意申请，-1 为拒绝申请 |
| handleMsg    | 1    | string | 处理的信息                  |
