### 请求参数示例

```json
{
  "fromUserID": "2824146701",
  "ToUserID": "3086243450",
  "reqMsg": "hello!",
  "ex": ""
}
```

| 字段名     | 选填 | 类型   | 说明                |
| :--------- | :--- | :----- | ------------------- |
| fromUserID | 必填 | string | 用户 ID             |
| ToUserID   | 必填 | string | 申请被添加的用户 ID |
| reqMsg     | 必填 | string | 申请信息            |
| ex         | 必填 | string | 拓展字段            |
