### 请求参数示例

```json

  "fromUserID": "2824146701",
  "ToUserID": "3086243450",
  "reqMsg": "hello!",
  "ex": ""
}
```

| 字段名     | 必选 | 类型   | 说明                |
| :--------- | :--- | :----- | ------------------- |
| fromUserID | 1    | string | 用户 ID             |
| ToUserID   | 1    | string | 申请被添加的用户 ID |

| ex         | 1    | string | 拓展字段            |
