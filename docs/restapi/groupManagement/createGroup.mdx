### 请求参数示例

```json
{
  "initMembers": ["1503849230", "1910095287"],
  "adminUserIDs": ["2769092611"],
  "ownerUserID": "199975690",
  "groupInfo": {
    "groupID": "xadxwr24",
    "groupName": "yourg group name",
    "notification": "notification",
    "introduction": "introduction",
    "faceURL": "faceURL url",
    "ex": "ex",
    "groupType": 0,
    "needVerification": 0,
    "lookMemberInfo": 0,
    "applyMemberFriend": 0
  }
}
```

| 字段名                      | 选填 | 类型   | 说明                     |
| :-------------------------- | :--- | :----- | ------------------------ |
| initMembers                 | 必填 | string | 初始化群成员 ID 列表     |
| adminUserIDs                | 必填 | array  | 管理用户用户 ID 列表     |
| ownerUserID                 | 必填 | string | 群主 ID                  |
| groupInfo                   | 必填 | object | 0 为普通群，2 为超级群   |
| groupInfo.groupID           | 选填 | string | 群 ID                    |
| groupInfo.groupName         | 必填 | string | 群名称                   |
| groupInfo.notification      | 必填 | string | 群通知                   |
| groupInfo.introduction      | 必填 | string | 群介绍                   |
| groupInfo.faceURL           | 必填 | string | 群头像                   |
| groupInfo.ex                | 必填 | string | 群拓展字段               |
| groupInfo.groupType         | 必填 | string | 群类型                   |
| groupInfo.needVerification  | 必填 | string | 进群是否需要验证         |
| groupInfo.lookMemberInfo    | 必填 | string | 是否能查看其他群成员信息 |
| groupInfo.applyMemberFriend | 必填 | string | 群成员是否能添加好友     |
