### 成功返回示例的参数说明

| 参数名                                 | 类型   | 说明                         |
| :------------------------------------- | :----- | :--------------------------- |
| errCode                                | int    | 错误码,0 表示成功            |
| errMsg                                 | string | 错误简要信息,无错误时为空    |
| errDlt                                 | errDlt | 错误详细信息,无错误时为空    |
| data                                   | object | 通用数据对象，具体结构见下方 |
| groupAbstractInfos                     | array  |                              |
| groupAbstractInfos.groupID             | string | 群 ID                        |
| groupAbstractInfos.groupMemberNumber   | string | 群人数                       |
| groupAbstractInfos.groupMemberListHash | string | 群成员哈希                   |
