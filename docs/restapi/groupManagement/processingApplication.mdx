| 字段名       | 选填 | 类型   | 说明                                           |
| :----------- | :--- | :----- | ---------------------------------------------- |
| groupID      | 必填 | string | 群 ID                                          |
| fromUserID   | 必填 | string | 申请用户 ID                                    |
| handledMsg   | 选填 | string | 处理信息                                       |
| handleResult | 必填 | string | 处理结果 -1 为拒绝该用户加群，1 为同意用户加群 |
