### 请求参数示例

```json
{
  "members": [
    {
      "groupID": "2137448827",
      "userID": "2699373280",
      "nickName": "newName",
      "faceURL": "new faceURL",
      "roleLevel": 60,
      "ex": ""
    }
  ]
}
```

| 字段名            | 选填 | 类型   | 说明                                       |
| :---------------- | :--- | :----- | ------------------------------------------ |
| members           | 必填 | array  | 要修改的群成员对象                         |
| members.groupID   | 必填 | string | 群 ID                                      |
| members.userID    | 必填 | string | 群成员用户 ID                              |
| members.nickName  | 选填 | string | 群成员昵称                                 |
| members.faceURL   | 选填 | string | 群成员头像                                 |
| members.roleLevel | 选填 | string | 群成员等级，1 为普通，2 为群主，3 为管理员 |
| members.ex        | 选填 | string | 群成员拓展字段                             |
