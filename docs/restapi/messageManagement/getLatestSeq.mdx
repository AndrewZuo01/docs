## 获取最新seq

</center>

### 简要描述
- 获取用户写扩散的最大最小seq，获取用户读扩散群最大最小seq
#### 接口状态
- 已完成
### 请求URL
- `post`
### 请求方式
- `{{host}}/msg/newest_seq`

### 请求参数示例


```json
{
"userID":"10000"
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|groupIDs|必填|array|用户加入的群ID列表，内部为字符串类型|
|userID|必填|string|用户ID|
### 成功返回示例


```json
{
  "errCode": 0,
  "errMsg": "",
  "errDlt": "",
  "data": {
    "maxSeq": 359,
    "minSeq": 1,
    "groupMaxAndMinSeq": {
      "1976917203": {
        "maxSeq": 0,
        "minSeq": 0
      }
    }
  }
}
```
### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
|maxSeq|string|用户最大的seq|
|minSeq|string|用户最小的seq|
|groupMaxAndMinSeq|object|群最大最小seq对象|
|groupMaxAndMinSeq.maxSeq|string|群最大seq|
|groupMaxAndMinSeq.minSeq|string|群最小seq|
