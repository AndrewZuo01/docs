## 获取最新 seq

</center>

### 简要描述

- 获取用户写扩散的最大最小 seq，获取用户读扩散群最大最小 seq

#### 接口状态

- 已完成

### 请求方法

- `post`

### 请求 URL

- `{{host}}/msg/newest_seq`

### 成功返回示例

```json
{
  "errCode": 0,
  "errMsg": "",
  "errDlt": "",
  "data": {
    "maxSeq": 359,
    "minSeq": 1,
    "groupMaxAndMinSeq": {
      "1976917203": {
        "maxSeq": 0,
        "minSeq": 0
      }
    }
  }
}
```

### 成功返回示例的参数说明

| 参数名                   | 类型   | 说明                         |
| :----------------------- | :----- | :--------------------------- |
| errCode                  | int    | 错误码,0 表示成功            |
| errMsg                   | string | 错误简要信息,无错误时为空    |
| errDlt                   | errDlt | 错误详细信息,无错误时为空    |
| data                     | object | 通用数据对象，具体结构见下方 |
| maxSeq                   | string | 用户最大的 seq               |
| minSeq                   | string | 用户最小的 seq               |
| groupMaxAndMinSeq        | object | 群最大最小 seq 对象          |
| groupMaxAndMinSeq.maxSeq | string | 群最大 seq                   |
| groupMaxAndMinSeq.minSeq | string | 群最小 seq                   |
