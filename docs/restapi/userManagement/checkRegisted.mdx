## 查询用户是否注册

</center>

### 简要描述

- 查询 userID 是否已经在 IM 系统中注册，应在 IM 注册之前调用，确保 userID 不会重复注册。

### 请求方式

- `post`

### 请求 URL

- `{{host}}/user/account_check`

### 成功返回示例

```json
{
  "errCode": 0,
  "errMsg": "",
  "errDlt": "",
  "data": {
    "results": [
      {
        "userID": "9906953281",
        "accountStatus": "registered"
      },
      {
        "userID": "9906953282",
        "accountStatus": "unregistered"
      }
    ]
  }
}
```

### 成功返回示例的参数说明

| 参数名                | 类型   | 说明                         |
| :-------------------- | :----- | :--------------------------- |
| errCode               | int    | 错误码,0 表示成功            |
| errMsg                | string | 错误简要信息,无错误时为空    |
| errDlt                | errDlt | 错误详细信息,无错误时为空    |
| data                  | object | 通用数据对象，具体结构见下方 |
| results               | array  |                              |
| results.userID        | string |                              |
| results.accountStatus | string |                              |

### 失败返回示例

```json
{
  "errCode": 1001,
  "errMsg": "ArgsError",
  "errDlt": "userID repeated[1001]ArgsError"
}
```

### 失败返回示例的参数说明

| 参数名  | 类型   | 说明                          |
| :------ | :----- | :---------------------------- |
| errCode | int    | 错误码,具体查看全局错误码文档 |
| errMsg  | string | 错误简要信息                  |
| errDlt  | errDlt | 错误详细信息                  |
