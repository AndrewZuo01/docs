### 成功返回示例

```json
{
  "errCode": 0,
  "errMsg": "",
  "errDlt": "",
  "data": {
    "usersInfo": [
      {
        "userID": "9906953281",
        "nickname": "chao",
        "faceURL": "http://203.56.175.233:10002/third/object?name=%2Fdata%2Fuser%2F0%2Fcn.rentsoft.flutter.openim.consumer%2Fcache%2Fimage_cropper_1687849862972.jpg",
        "ex": "",
        "createTime": 0,
        "appMangerLevel": 18,
        "globalRecvMsgOpt": 0
      }
    ]
  }
}
```

### 成功返回示例的参数说明

| 参数名                 | 类型   | 说明                                   |
| :--------------------- | :----- | :------------------------------------- |
| total                  | string | 用户总数                               |
| users                  | array  | 用户列表                               |
| users.userID           | string | 用户 ID                                |
| users.nickname         | string | 用户名                                 |
| users.faceURL          | string | 用户头像                               |
| users.ex               | string | 拓展字段                               |
| users.createTime       | string | 创建时间                               |
| users.appMangerLevel   | string | 管理员等级                             |
| users.globalRecvMsgOpt | number | 全局是否接收消息，0 为接收，1 为不接收 |

### 失败返回示例

```json
{
  "errCode": 1001,
  "errMsg": "ArgsError",
  "errDlt": ""
}
```

### 失败返回示例的参数说明

| 参数名  | 类型   | 说明                          |
| :------ | :----- | :---------------------------- |
| errCode | int    | 错误码,具体查看全局错误码文档 |
| errMsg  | string | 错误简要信息                  |
| errDlt  | errDlt | 错误详细信息                  |
