## 取消订阅用户的在线状态

</center>

### 简要描述
- 取消订阅一批用户的在线状态
#### 接口状态
- 已完成
### 请求方式
- `post` 
### 请求URL
- `{{host}}//user/unsubscribe_users_status` 


### Header
|header名|示例值|选填|类型|说明|
|:----    |:-------    |:--- |---|------      |
|operationID|asdad3qds1|必填|string|operationID用于全局链路追踪|
|token|{{token}}|必填|string|管理员或者用户token|


### 请求参数示例


```json
{
    "userID": "9168684795",
    "userIDs": [
        "3436560733",
        "6317136453",
        "8450535746"
    ],
    "genre": 2
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|userID|必填|string|用户账号|
|userIDs|必填|array|取消订阅的用户列表|
|genre|必填|string|订阅或取消订阅的枚举，1 - 订阅； 2  - 取消订阅|
### 成功返回示例


```json
{
    "errCode": 0,
    "errMsg": "",
    "errDlt": "",
    "data": {
        "statusList": null
    }
}
```
### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
|statusList|string|成功无数据返回|
