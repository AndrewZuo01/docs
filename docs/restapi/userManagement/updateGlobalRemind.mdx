## 设置用户全局免打扰

</center>

### 简要描述

- 设置用户全局免打扰，所有消息的接收属于静默状态，无推送提醒。

### 请求方式

- `post`

### 请求 URL

- `{{host}}/user/set_global_msg_recv_opt`

### 请求参数示例

```json
{
  "userID": "11111112",
  "globalRecvMsgOpt": 0
}
```

| 字段名           | 选填 | 类型   | 说明 |
| :--------------- | :--- | :----- | ---- |
| userID           | 必填 | string |      |
| globalRecvMsgOpt | 必填 | string |      |

### 成功返回示例

```json
{
  "errCode": 0,
  "errMsg": "",
  "errDlt": "",
  "data": {}
}
```

### 成功返回示例的参数说明

| 参数名  | 类型   | 说明                         |
| :------ | :----- | :--------------------------- |
| errCode | int    | 错误码,0 表示成功            |
| errMsg  | string | 错误简要信息,无错误时为空    |
| errDlt  | errDlt | 错误详细信息,无错误时为空    |
| data    | object | 通用数据对象，具体结构见下方 |

### 失败返回示例

```json
{
  "errCode": 1004,
  "errMsg": "RecordNotFoundError",
  "errDlt": "userID not found[1004]RecordNotFoundError"
}
```

### 失败返回示例的参数说明

| 参数名  | 类型   | 说明                          |
| :------ | :----- | :---------------------------- |
| errCode | int    | 错误码,具体查看全局错误码文档 |
| errMsg  | string | 错误简要信息                  |
| errDlt  | errDlt | 错误详细信息                  |
