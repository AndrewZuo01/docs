### **简要描述**

- 这里是发送消息前的 webhook 调用例子。

## ⚙️ 配置说明

```yaml
callback:
  url: "http://www.example.com"
  beforeSendSingleMsg:
    enable: true # true 代表开启改回调
    timeout: 5 # 响应超时时间
    failedContinue: false # false 代表回调失败后直返返回报错，停止继续往下执行
  afterSendSingleMsg:
    enable: true
    timeout: 5
    failedContinue: false
```

## 🔄 回调执行过程说明

启动 **url** 对应的服务。按如上内容要求修改好对应的配置后，在 **open-im-server** 项目下 **make build** 和 **make start** 命令，**make check** 检查无误后。
可以发送消息进行验证
