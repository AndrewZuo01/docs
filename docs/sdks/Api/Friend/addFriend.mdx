# addFriend

## 功能介绍

:::info 说明

发起好友申请，请求添加对方为好友

:::

:::caution 注意

主动添加者和被添加者都会收到 onFriendApplicationAdded 回调，开发者根据回调中发起者 userID 和登录用户 userID 来判断是主动添加还是被动添加

:::

 Future<dynamic> addFriend({

| 参数名称 | 参数类型 | 是否必填 | 描述     |

| uid      | String   | 是       | 用户 ID  |

| 参数名称 | 参数类型 | 描述                 |
