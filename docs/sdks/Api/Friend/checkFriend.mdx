# checkFriend

## 功能介绍

:::info 说明

检测好友关系，判断是否在已登录用户的好友列表中

:::

:::caution 注意

由于好友关系是双向关系，仅检测对方是否在自己的好友列表中，并不能检测自己是否在对方的好友列表中

:::
