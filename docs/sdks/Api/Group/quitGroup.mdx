# quitGroup

## 功能介绍

:::info 说明

退出群聊

:::

:::caution 注意

（1）群主不能退出群聊，需要在退出群聊之前先转让群主身份；  
（2）退出者收到 OnJoinedGroupDeleted 回调，其他群成员收到 OnGroupMemberDeleted 回调。

:::

### 函数原型

```swift showLineNumbers
- (void)quitGroup:(NSString *)groupID
```

```swift showLineNumbers

}];
```

```ts showLineNumbers
IMSDK.quitGroup(groupID: string, operationID?: string): Promise<WsResponse>
```

```js showLineNumbers
const IMSDK = getSDK();

  .catch(({ errCode, errMsg }) => {
    // 调用失败
  });
```

<TabItem value="uni-app">

### 函数原型

```ts showLineNumbers
IMSDK.asyncApi('quitGroup', operationID: string, groupID: string): Promise<void>
```

### 输入参数

| 参数名称    | 参数类型 | 是否必填 | 描述                                                    |
| ----------- | -------- | -------- | ------------------------------------------------------- |
| operationID | string   | 是       | 操作 ID，用于定位问题，保持唯一，建议用当前时间和随机数 |
| groupID     | string   | 是       | 群聊 ID                                                 |

### 返回结果

> 通过`openim-uniapp-polyfill`包将函数 Promise 化，调用时需要使用`then`和`catch`判断并处理成功和失败回调。

| 参数名称        | 参数类型                                                         | 描述         |
| --------------- | ---------------------------------------------------------------- | ------------ |
| Promise.then()  | Promise<void\>                                                   | 调用成功回调 |
| Promise.catch() | Promise<[CatchResponse](docs/sdks/Class/Response/Response.mdx)\> | 调用失败回调 |

### 代码示例

```js showLineNumbers
import IMSDK from 'openim-uniapp-polyfill';

IMSDK.asyncApi('quitGroup', Date.now(), 'groupID')
  .then(() => {
    // 调用成功
  })
  .catch(({ errCode, errMsg }) => {
    // 调用失败
  });
```
