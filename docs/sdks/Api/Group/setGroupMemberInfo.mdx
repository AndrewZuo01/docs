# setGroupMemberInfo

## 功能介绍

:::info 说明

修改群成员信息

:::

:::caution 注意

群主可以修改管理员和普通成员信息；管理员可以修改普通成员信息；  
群成员会收到 OnGroupMemberInfoChanged 回调；

:::
