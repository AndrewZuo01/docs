# setGroupVerification

## 功能介绍

:::info 说明

设置进群验证方式

:::

:::caution 注意

管理员和群主才有设置权限

:::

### 函数原型

```dart showLineNumbers
  Future<dynamic> setGroupVerification({
    required String groupID,
    required int needVerification,
    String? operationID,
  })
```

### 输入参数

| 参数名称         | 参数类型                                                         | 是否必填 | 描述     |
| ---------------- | ---------------------------------------------------------------- | -------- | -------- |
| groupID          | String                                                         | 是       | 群 ID    |
| needVerification | int | 是       | 进群设置[GroupVerification](docs/sdks/Enum/GroupVerification.mdx) |

### 返回结果

| 参数名称  | 参数类型                                               | 描述     |
| --------- | ------------------------------------------------------ | -------- |
| ~ | ~ | 无异常则操作成功 |

### 代码示例

```dart showLineNumbers
   await OpenIM.iMManager.groupManager.setGroupVerification(
      groupID: '',
      needVerification: GroupVerification.allNeedVerification,
    );
    // todo
```

### 函数原型

```swift showLineNumbers

- (void)setGroupVerification:(NSString *)groupID
            needVerification:(OIMGroupVerificationType)needVerification
                   onSuccess:(nullable OIMSuccessCallback)onSuccess
                   onFailure:(nullable OIMFailureCallback)onFailure;

```

### 输入参数

| 参数名称         | 参数类型                                                         | 是否必填 | 描述     |
| ---------------- | ---------------------------------------------------------------- | -------- | -------- |
| groupID          | NSString                                                         | 是       | 群 ID    |
| needVerification | [OIMGroupVerificationType](docs/sdks/Enum/GroupVerification.mdx) | 是       | 进群设置 |

### 返回结果

| 参数名称  | 参数类型                                               | 描述     |
| --------- | ------------------------------------------------------ | -------- |
| onSuccess | [OIMSuccessCallback](docs/sdks/Callback/onSuccess.mdx) | 成功返回 |
| onFailure | [OIMFailureCallback](docs/sdks/Callback/onError.mdx)   | 失败返回 |

### 代码示例

```swift showLineNumbers

[OIMManager.manager setGroupVerification:@""
                        needVerification:OIMGroupVerificationTypeDirectly
                               onSuccess:^(NSString * _Nullable data) {
} onFailure:^(NSInteger code, NSString * _Nullable msg) {
}];

```
