# getConnectStatus

## 功能介绍

:::info 说明

获取用户连接状态

:::

:::caution 注意

此接口主要用于 UI 显示，由于 SDK 内部有重试机制，不要根据连接状态做任何逻辑操作，如不要因为连接失败而再调次用 login。

:::

<TabItem value="Web">

### 函数原型

```ts showLineNumbers
IMSDK.getConnectStatus(): Promise<WsResponse<ConnectStatus>>;
```

### 输入参数

无

### 返回结果

| 参数名称        | 参数类型                                                                | 描述         |
| --------------- | ----------------------------------------------------------------------- | ------------ |
| Promise.then()  | Promise<WsResponse<[ConnectStatus](docs/sdks/Enum/ConnectStatus.mdx)>\> | 链接状态     |
| Promise.catch() | Promise<[WsResponse](docs/sdks/Class/Response/Response.mdx)\>           | 调用失败回调 |

### 代码示例

```js showLineNumbers
import { getSDK } from 'open-im-sdk-wasm';
const IMSDK = getSDK();

const status = IMSDK.getConnectStatus()
  .then(({ data }) => {
    // data: 链接状态ConnectStatus
  })
  .catch(({ errCode, errMsg }) => {
    // 调用失败
  });
```
