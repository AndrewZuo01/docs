# setAppBackgroundStatus

## 功能介绍

:::info 说明

设置 app 是否后台运行，用户把 app 在前后台之前切换时调用

:::

:::caution 注意

app 端需要正确设置，确保离线实时到达

:::

### 函数原型

```swift showLineNumbers

- (void)setAppBackgroundStatus:(BOOL)isBackground
                     onSuccess:(nullable OIMSuccessCallback)onSuccess
                     onFailure:(nullable OIMFailureCallback)onFailure;

```

### 输入参数

| 参数名称     | 参数类型 | 是否必填 | 描述         |
| ------------ | -------- | -------- | ------------ |
| isBackground | BOOL     | 是       | 是否切到后台 |

### 返回结果

| 参数名称  | 参数类型                                                     | 描述     |
| --------- | ------------------------------------------------------------ | -------- |
| onSuccess | [OIMSuccessCallback](docs/sdks/Callback/SuccessCallback.mdx) | 成功返回 |
| onFailure | [OIMFailureCallback](docs/sdks/Callback/ErrorCallback.mdx)   | 失败返回 |

### 代码示例

```swift showLineNumbers

[OIMManager.manager setAppBackgroundStatus:YES
                                     onSuccess:^(NSString * _Nullable data) {
} onFailure:^(NSInteger code, NSString * _Nullable msg) {
}];

```

<TabItem value="Web">
> 主要用于Electron，从托盘打开应用时，可以调用此方法，快速触发重连。

### 函数原型

```ts showLineNumbers

IMSDK.setAppBackgroundStatus(isBackground: boolean): Promise<WsResponse>;

```

### 输入参数

| 参数名称     | 参数类型 | 是否必填 | 描述         |
| ------------ | -------- | -------- | ------------ |
| isBackground | boolean  | 是       | 是否切到后台 |

### 返回结果

| 参数名称        | 参数类型                                                      | 描述         |
| --------------- | ------------------------------------------------------------- | ------------ |
| Promise.then()  | Promise<[WsResponse](docs/sdks/Class/Response/Response.mdx)\> | 调用成功回调 |
| Promise.catch() | Promise<[WsResponse](docs/sdks/Class/Response/Response.mdx)\> | 调用失败回调 |

### 代码示例

```js showLineNumbers
import { getSDK } from 'open-im-sdk-wasm';
const IMSDK = getSDK();

const status = IMSDK.setAppBackgroundStatus(true)
  .then(() => {
    // 调用成功
  })
  .catch(({ errCode, errMsg }) => {
    // 调用失败
  });
```

<TabItem value="uni-app">

### 函数原型

```ts showLineNumbers
IMSDK.asyncApi('setAppBackgroundStatus', operationID: string,isBackground: boolean): Promise<void>
```

### 输入参数

| 参数名称     | 参数类型 | 是否必填 | 描述                                                    |
| ------------ | -------- | -------- | ------------------------------------------------------- |
| operationID  | string   | 是       | 操作 ID，用于定位问题，保持唯一，建议用当前时间和随机数 |
| isBackground | boolean  | 是       | 是否切到后台                                            |

### 返回结果

> 通过`openim-uniapp-polyfill`包将函数 Promise 化，调用时需要使用`then`和`catch`判断并处理成功和失败回调。

| 参数名称        | 参数类型                                                         | 描述         |
| --------------- | ---------------------------------------------------------------- | ------------ |
| Promise.then()  | Promise<void\>                                                   | 调用成功回调 |
| Promise.catch() | Promise<[CatchResponse](docs/sdks/Class/Response/Response.mdx)\> | 调用失败回调 |

### 代码示例

```js showLineNumbers
import IMSDK from 'openim-uniapp-polyfill';

IMSDK.asyncApi('setAppBackgroundStatus', Date.now(), true)
  .then(() => {
    // 调用成功
  })
  .catch(({ errCode, errMsg }) => {
    // 调用失败
  });
```
