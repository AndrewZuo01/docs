### 函数原型

```dart showLineNumbers
 Future<Message> createImageMessage({
    required String imagePath,
    String? operationID,
  })
```

### 输入参数

| 参数名称  | 参数类型 | 是否必填 | 描述                                                                                                                         |
| --------- | -------- | -------- | ---------------------------------------------------------------------------------------------------------------------------- |
| imagePath | String | 是       | initSDK 时传入了数据缓存路径，如路径：A，这时需要你将图片复制到 A 路径下后，如 A/pic/a.png 路径，imagePath 的值：“/pic/.png” |

### 返回结果

| 名称    | 类型                                                      | 描述     |
| ------- | --------------------------------------------------------- | -------- |
| ~ | [Message](docs/sdks/Class/Message/MessageInfo.mdx) | 成功返回 |
