### 函数原型

```swift showLineNumbers

+ (OIMMessageInfo *)createMergeMessage:(NSArray <OIMMessageInfo *> *)messages
                                 title:(NSString *)title
                           summaryList:(NSArray <NSString *> *)summarys;

```

### 输入参数

| 参数名称 | 参数类型                                                                 | 是否必填 | 描述         |
| -------- | ------------------------------------------------------------------------ | -------- | ------------ |
| messages | NSArray < [OIMMessageInfo](docs/sdks/Class/Message/MessageInfo.mdx) \* > | 是       | 消息列表     |
| title    | NSString                                                                 | 是       | 标题         |
| summarys | NSArray <NSString \*>                                                    | 是       | 消息摘要列表 |

### 返回结果

| 名称    | 类型                                                      | 描述     |
| ------- | --------------------------------------------------------- | -------- |
| message | [OIMMessageInfo](docs/sdks/Class/Message/MessageInfo.mdx) | 成功返回 |

### 代码示例

```swift showLineNumbers

OIMMessageInfo *message = [OIMMessageInfo createMergeMessage:@[]
                                                       title:@""
                                                 summaryList:@[]];

```
