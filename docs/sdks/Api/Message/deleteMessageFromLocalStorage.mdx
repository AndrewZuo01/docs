## 功能介绍

:::info 说明

:::

:::caution 注意

```dart showLineNumbers
  Future deleteMessageFromLocalStorage({
    required String conversationID,
    required String clientMsgID,

  })
```

### 输入参数

| conversationID  | String |   是     | 回话ID |
