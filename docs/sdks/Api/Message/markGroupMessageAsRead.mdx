### 函数原型

```dart showLineNumbers
  Future markGroupMessageAsRead({
    required String groupID,
    required List<String> messageIDList,
    String? operationID,
  })
```

### 输入参数

| 参数名称  | 参数类型              | 是否必填 | 描述               |
| --------- | --------------------- | -------- | ------------------ |
| groupID   | String              | 是       | 群 ID              |
| messageIDList | List<String\> | 是       | clientMsgID 的列表 |

### 返回结果

| 名称      | 类型                                                   | 描述     |
| --------- | ------------------------------------------------------ | -------- |
| ~ | ~ | 无异常抛出则操作成功 |

    await OpenIM.iMManager.messageManager.markGroupMessageAsRead(groupID: '', messageIDList: []);

### 函数原型

```swift showLineNumbers

- (void)markGroupMessageAsRead:(NSString *)groupID
                     msgIDList:(NSArray <NSString *> *)msgIDList
                     onSuccess:(nullable OIMSuccessCallback)onSuccess
                     onFailure:(nullable OIMFailureCallback)onFailure;

```

### 输入参数

| 参数名称  | 参数类型              | 是否必填 | 描述               |
| --------- | --------------------- | -------- | ------------------ |
| groupID   | NSString              | 是       | 群 ID              |
| msgIDList | NSArray <NSString \*> | 是       | clientMsgID 的列表 |

### 返回结果

| 名称      | 类型                                                   | 描述     |
| --------- | ------------------------------------------------------ | -------- |
| onSuccess | [OIMSuccessCallback](docs/sdks/Callback/onSuccess.mdx) | 成功返回 |
| onFailure | [OIMFailureCallback](docs/sdks/Callback/onError.mdx)   | 失败返回 |

### 代码示例

```swift showLineNumbers

[OIMManager.manager markGroupMessageAsRead:@""
                                 msgIDList:@[]
                                 onSuccess:^(NSString * _Nullable data) {
} onFailure:^(NSInteger code, NSString * _Nullable msg) {
}];

```
