### 函数原型

```dart showLineNumbers
  Future<dynamic> setGlobalRecvMessageOpt({
    required int status,
    String? operationID,
  })
```

### 输入参数

| 参数名称 | 参数类型                                              | 是否必填 | 描述 |
| -------- | ----------------------------------------------------- | -------- | ---- |
| status      | int | 是       |   0：正常；1：不接受消息；2：接受在线消息不接受离线消息；   |

### 返回结果

| 名称      | 类型                                                   | 描述     |
| --------- | ------------------------------------------------------ | -------- |
| ~ | ~ | 无异常抛出则操作成功 |
