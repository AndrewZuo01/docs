### OIMGroupInfo

| 字段名称               | 字段类型                                                         | 描述                     |
| ---------------------- | ---------------------------------------------------------------- | ------------------------ |
| groupID                | NSString                                                         | 群唯一 ID                |
| groupName              | NSString                                                         | 群名称                   |
| notification           | NSString                                                         | 群公告                   |
| introduction           | NSString                                                         | 群简介                   |
| faceURL                | NSInteger                                                        | 群头像链接               |
| createTime             | NSInteger                                                        | 群创建时间               |
| status                 | [OIMGroupStatus](docs/sdks/Enum/GroupStatus.mdx)                 | 群状态                   |
| creatorUserID          | NSString                                                         | 创建者 userID            |
| groupType              | [OIMGroupType](docs/sdks/Enum/GroupType.mdx)                     | 群状态                   |
| ownerUserID            | NSString                                                         | 群主 userID              |
| memberCount            | NSInteger                                                        | 群成员数量               |
| ex                     | NSString                                                         | 应用层扩展字段           |
| needVerification       | [OIMGroupVerificationType](docs/sdks/Enum/GroupVerification.mdx) | 进群验证方式             |
| lookMemberInfo         | NSInteger                                                        | 是否允许查看群成员信息   |
| applyMemberFriend      | NSInteger                                                        | 是否可以添加群成员为好友 |
| notificationUpdateTime | NSInteger                                                        | 群公告更新时间           |
| notificationUserID     | NSString                                                         | 群公告发布者 userID      |
