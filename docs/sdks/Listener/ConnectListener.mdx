# ConnectListener

:::info

SDK 连接状态回调

:::

| 模块名称           | 模块功能简介                           |
| ------------------ | -------------------------------------- |
| onConnectFailed    | SDK 连接服务器失败                     |
| onConnectSuccess   | SDK 连接服务器成功                     |
| onConnecting       | SDK 正在连接服务器                     |
| onKickedOffline    | 账号已在其他地方登录，当前设备被踢下线 |
| onUserTokenExpired | 登录凭证过期，需要重新登录             |
