# GroupListener

| [onGroupApplicationAdded](docs/sdks/Callback/onGroupApplicationAdded.mdx)       | 新的群申请通知     |
| [onGroupApplicationDeleted](docs/sdks/Callback/onGroupApplicationDeleted.mdx)   | 群申请删除通知     |
| [onGroupApplicationRejected](docs/sdks/Callback/onGroupApplicationRejected.mdx) | 群申请被拒绝通知   |
