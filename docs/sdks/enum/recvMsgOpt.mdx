# RecvMsgOpt

:::info

会话是否接收离线消息推送

:::

## 会话消息接收选项

| 枚举值 | 描述                   |
| ------ | ---------------------- |
| 0      | 正常接受消息 |
| 1      |   保留字段  |
| 2      |     此会话接收消息，但无离线推送（即会话免打扰） |
