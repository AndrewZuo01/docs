<TabItem value="React-Native">

### 函数原型

```ts showLineNumbers
OpenIMSDKRN.getFriendList(
  operationID: string
);

```

### 输入参数

| 参数名称    | 参数类型 | 是否必填 | 描述                                                    |
| ----------- | -------- | -------- | ------------------------------------------------------- |
| operationID | string   | 是       | 操作 ID，用于定位问题，保持唯一，建议用当前时间和随机数 |

### 返回结果

> 通过`openim-uniapp-polyfill`包将函数 Promise 化，调用时需要使用`then`和`catch`判断并处理成功和失败回调。

| 参数名称        | 参数类型                                                            | 描述             |
| --------------- | ------------------------------------------------------------------- | ---------------- |
| Promise.then()  | Promise<[FriendUserItem](docs/sdks/class/friend/friendInfo.mdx)[]\> | 好友信息对象列表 |
| Promise.catch() | Promise<[CatchResponse](docs/sdks/class/response.mdx)\>             | 调用失败回调     |

### 代码示例

```js showLineNumbers
import OpenIMSDKRN from "open-im-sdk-rn";

try {
  const operationID = 'OperationID_value';

  const data = await OpenIMSDKRN.getFriendList(operationID);

  console.log("getFriendList: Retrieved friend list", data);
} catch (error) {
  console.error('Error getFriendList:', error);
}

```
