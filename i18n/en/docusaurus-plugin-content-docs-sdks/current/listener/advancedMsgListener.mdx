# AdvancedMsgListener

## 模块概览

:::info

消息监听器，当收到新消息、已读回执及撤回消息时回调

:::

| 模块名称                                                                  | 模块功能简介     |
| ------------------------------------------------------------------------- | ---------------- |
| [onRecvNewMessage](/callback/onRecvNewMessage.mdx)               | 收到新消息       |
| [onRecvOfflineNewMessage](/callback/onRecvOfflineNewMessage.mdx) | 收到离线消息     |
| [onRecvC2CReadReceipt](/callback/onRecvC2CReadReceipt.mdx)       | 单聊消息已读回执 |
| [onRecvGroupReadReceipt](/callback/onRecvGroupReadReceipt.mdx)   | 群聊消息已读回执 |
| [onNewRecvMessageRevoked](/callback/onNewRecvMessageRevoked.mdx) | 撤回消息回调     |
