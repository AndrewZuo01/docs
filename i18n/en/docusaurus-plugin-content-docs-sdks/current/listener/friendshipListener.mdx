# RelationshipListener

## 模块概览

:::info

好友及黑名单关系链监听器，当好友及黑名单信息改变时回调。

:::

| 模块名称                                                                          | 模块功能简介     |
| --------------------------------------------------------------------------------- | ---------------- |
| [onFriendApplicationAdded](/callback/onFriendApplicationAdded.mdx)       | 好友申请新增通知 |
| [onFriendApplicationRejected](/callback/onFriendApplicationRejected.mdx) | 好友申请被拒绝   |
| [onFriendApplicationAccepted](/callback/onFriendApplicationAccepted.mdx) | 好友申请被接受   |
| [onFriendApplicationDeleted](/callback/onFriendApplicationDeleted.mdx)   | 好友申请被删除   |
| [onFriendAdded](/callback/onFriendAdded.mdx)                             | 好友新增通知     |
| [onFriendDeleted](/callback/onFriendDeleted.mdx)                         | 好友删除通知     |
| [onFriendInfoChanged](/callback/onFriendInfoChanged.mdx)                 | 好友资料变更通知 |
| [onBlackAdded](/callback/onBlackAdded.mdx)                               | 黑名单新增通知   |
| [onBlackDeleted](/callback/onBlackDeleted.mdx)                           | 黑名单删除通知   |
