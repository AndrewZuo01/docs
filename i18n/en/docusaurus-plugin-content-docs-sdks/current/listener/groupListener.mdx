# GroupListener

## 模块概览

:::info

群组监听器，当群组及群成员信息改变时回调。

:::

| 模块名称                                                                        | 模块功能简介       |
| ------------------------------------------------------------------------------- | ------------------ |
| [onGroupMemberAdded](/callback/onGroupMemberAdded.mdx)                 | 有新成员加入群     |
| [onGroupMemberDeleted](/callback/onGroupMemberDeleted.mdx)             | 有成员离开群       |
| [onGroupMemberInfoChanged](/callback/onGroupMemberInfoChanged.mdx)     | 某成员信息发生变更 |
| [onJoinedGroupAdded](/callback/onJoinedGroupAdded.mdx)                 | 群成员变更         |
| [onJoinedGroupDeleted](/callback/onJoinedGroupDeleted.mdx)             | 群成员变更         |
| [onGroupInfoChanged](/callback/onGroupInfoChanged.mdx)                 | 群信息变更         |
| [onGroupApplicationAccepted](/callback/onGroupApplicationAccepted.mdx) | 群申请被接受通知   |
| [onGroupApplicationAdded](/callback/onGroupApplicationAdded.mdx)       | 新的群申请通知     |
| [onGroupApplicationDeleted](/callback/onGroupApplicationDeleted.mdx)   | 群申请删除通知     |
| [onGroupApplicationRejected](/callback/onGroupApplicationRejected.mdx) | 群申请被拒绝通知   |
| [onGroupDismissed](/callback/onGroupDismissed.mdx) | 群解散通知   |
